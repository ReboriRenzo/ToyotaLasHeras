# Toyota Las Heras

Sitio web modernizado para Toyota Las Heras - Repuestos y accesorios Toyota y Volkswagen.

## Características

- **Diseño moderno** con tema oscuro y acentos en rojo Toyota
- **100% responsive** para móvil, tablet y desktop
- **Heroicons** para iconografía consistente
- **Animaciones suaves** al hacer scroll
- **Formulario de contacto** con redirección a WhatsApp
- **Accesibilidad** con etiquetas ARIA y navegación por teclado

## Estructura

```
ToyotaLasHeras/
├── index.html    # Página principal
├── styles.css    # Estilos
├── script.js     # Interactividad
└── README.md
```

## Cómo usar

1. Abrí `index.html` en tu navegador
2. O usá un servidor local: `npx serve .`

## Recursos utilizados

- [Heroicons](https://heroicons.com/) - Iconos SVG
- [Plus Jakarta Sans](https://fonts.google.com/specimen/Plus+Jakarta+Sans) - Tipografía
- Inspiración de [FreeFrontend](https://freefrontend.com/)
